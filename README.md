# Personal Portfolio Website ✨

Welcome to my personal portfolio website! This repository contains the full source code and structure for my personal portfolio, designed to showcase my academic projects, skills, and background in Information Technology.

---

## 🌟 Overview
The project is a clean, modern, and fully responsive single-page portfolio built using pure **HTML5** and **CSS3**. It includes custom styling, interactive project showcases, and responsive design elements.

---

## 🛠️ Built With
* **HTML5**: Semantic document structure.
* **CSS3**: Custom styling, CSS Flexbox & Grid layouts, dynamic gradients, and animations.
* **Google Fonts**: Plus Jakarta Sans typography.
* **Font Awesome**: Modern visual iconography.

---

## 🚀 Featured Sections
1. **Header & Navigation**: Smooth scrolling and clean layout.
2. **Hero Banner**: Personal introduction and key focus areas (*Cybersecurity & Web Development*).
3. **Professional Overview**: Detailed background and core competencies.
4. **Key Projects & Portfolio**: Interactive project cards showcasing practical software engineering and web platforms.
5. **Connect Section**: Direct links to GitHub repositories and email contact.

---

## 📁 Repository Structure
* **index.html**: Main HTML structure with embedded CSS styling.
* **README.md**: Project documentation and overview.
